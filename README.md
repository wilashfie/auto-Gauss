# auto-Gauss

Routine to fit six Gaussians to a Si IV 1402.77 <sup>Å</sup> spectral line profile. Accounts for the following components:

1. Main (central) Emission
2. Red-wing Emisison 
3. Stationary Emission
4. Stationary Absorbtion 
5. Fe <sup>II</sup> at 1403.10 <sup>Å</sup>
6. Fe <sup>II</sup> at 1403.13 <sup>Å</sup>

Fit will return best combination of these six-Gaussians.
